Projects

Private Wall

The Private Wall application is a user friendly web application that funtions as a private messaging platform between registered users and their friends network. It closely mirrors a traditional social media timeline such as 'Twitter'. This application was built using Python, Flask, and MySQL as the database.

Product Manager

A sorting application that allows for users to create products for sale and additionally, categories that each respective product would fall under to make searching for products much simpler. This application was built using Java, MySQL, and Spring Tool Suite.

Rate My Cake

Rate My Cake is a lightweight Angular application for managing ratings for user-submitted cakes. Additionally, cakes can be rated through a 1-5 star scale, comments, or both. The average star rating of each cake is highlighted above the cakes and is ordered in descending order. This application was built using the MEAN stack(MongoDB, Express, Angular, Nodejs).
